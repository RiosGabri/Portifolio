Adoção+

O Adoção+ é uma plataforma desenvolvida em Python utilizando Flask, HTML e CSS. O objetivo do sistema é facilitar o processo de adoção de animais, oferecendo um ambiente simples e intuitivo para cadastrar animais, visualizar perfis e gerenciar informações. O projeto foi pensado tanto para rodar no terminal quanto em uma interface web, permitindo maior flexibilidade no uso e contribuindo para a prática de construção de aplicações completas. Além disso, o Adoção+ reforça conceitos importantes como:

  • CRUD completo (criação, leitura, atualização e remoção)
  • Estruturação de rotas Flask
  • Organização de dados e boas práticas em back-end
  • Integração entre front-end simples e lógica Python
