Adoção+

O Adoção+ é um sistema desenvolvido utilizando Python, Flask, HTML e CSS, projetado para funcionar tanto no terminal quanto na interface web. Ele permite o cadastro de animais, gerenciamento de informações, visualização organizada dos dados e oferece um fluxo simples e educativo para estudantes compreenderem a integração entre backend e frontend.
